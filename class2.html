Variable Scope

The scope of a variable is how long the computer will remember it or have access to it.


JavaScript has function scope, meaning that the scope changes once you are inside of a function.

Global Scope

A variable declared outside a function has a global scope and can be accessed anywhere, even in a function.

  
    var awesomeGroup = 'Girl Develop It'; // Global scope
    function whatIsAwesome() {
      console.log (awesomeGroup + ' is pretty awesome.'); // Will work
    }
    whatIsAwesome(); // Calling the function

Local Scope

A variable declared within a function has a local scope and can only be accessed within that function.

  
    function whatIsAwesome() {
      var awesomeGroup = 'Girl Develop It'; // Local scope
      console.log ('I made a variable called awesomeGroup with a value 
      of ' + awesomeGroup); // Will work
    }
    
    whatIsAwesome(); // Calling the function
    console.log (awesomeGroup + ' is pretty awesome.'); // Won't work

Boolean Variables

Boolean variables represent the logical values true and false

  
    var catsAreBest = true;
    var dogsRule = false;

If you try to use another variable as a boolean, JavaScript will guess. The number 0, -0, an empty string '', undefined, NaN, null and false are considered false, everything else reads as true.

Important vocabulary to know

NaN = 'not a number';

var x = 100 / "Apple" // NaN


null = 'nothing' or 'empty' in JavaScript. It is a JavaScript object and means that no value is present.

The if statement

Use an if statement to execute a block of code if a condition is true.

  
    if (condition) {
      // statements to execute if condition is true
      // if condition is false, nothing happens
    }
  

  
    var bananas = 5;
    if (bananas > 0) {
      console.log('You have some bananas!');
    }

Comparison Operators

Example Name Result
a == b Equal TRUE if a is equal to b (can be different types).
a === b Identical TRUE if a is equal to b, and the same type.
a != b Not equal TRUE if a is not equal to b (can be different types).
a !== b Not identical TRUE if a is not equal to b, or they are not the same type.
a < b Less than TRUE if a is strictly less than b.
a > b Greater than TRUE if a is strictly greater than b.
a <= b Less than or equal to TRUE if a is less than or equal to b.
a >= b Greater than or equal to TRUE if a is greater than or equal to b.

Watch out!

Don't mix up =, == and ===


The single equals(=) is an assignment operator

var x = 3;

The double equals(==) checks if two statements are equal to each other, even though they do not have to have the same type. 

3 == '3'; // true, even though you are comparing a string and a number

The triple equals(===) checks if two statements are equal to each other and are the same type.

3 === '3'; // false, because a string and a number are different types.

The if/ else statement

Use else to provide an alternate/default set of instructions.

  
  var age = 28;
  if (age >= 16) {
      console.log ('Yay, you can drive!');
  } 
  else {
      console.log ('Sorry, but you have ' + (16 - age) +
      ' years until you can drive.');
  }

/* you don't need to add a condition to the else block because it 
is the default, it will run if the conditions above it are false */

The if/ else if/ else statement

If you have multiple conditions, you can use else if.

  
    var age = 20;
    if (age >= 35) {
        console.log('You can vote AND hold any place in government!');
    } else if (age >= 25) {
        console.log('You can vote AND run for the Senate!');
    } else if (age >= 18) {
        console.log('You can vote!'); // this is the condition that will run
    } else {
        console.log('You can\'t vote, but you can
        still write your representatives.');
    }

Switch Statement

You can use a switch statement to select one of many code blocks to be executed.

Syntax

  
    switch(condition) {
      case n:
        code block;
        break;
      case n:
        code block;
        break;
      default:
        default code block;
    }

Like an if/else statement, each case is compared to the condition. If there is a match, the associated code is executed.

    var day;
    switch (new Date().getDay()) {
    case 0:
        day = "Sunday"; break;
    case 1:
        day = "Monday"; break;
    case 2:
        day = "Tuesday"; break;
    case 3:
        day = "Wednesday"; break;
    case 4:
        day = "Thursday"; break;
    case 5:
        day = "Friday"; break;
    case 6:
        day = "Saturday"; break;
    }
    console.log('Yay! Its ' + day);

In JS, there is a Date() object. That object has a method called getDay().

The getDay() method returns the day of the week as a number between 0 and 6. (Sunday=0, Monday=1, Tuesday=2 ...)

Code Block

A code block is a block of code between two curly braces.

Break Keyword

Once the switch statement finds a true condition, you should exit out of the code block, using the break keyword.


Warning:

If you don't use a 'break' keyword at the end of each code block, it will run all of the statements that follow the 'true' condition.

Default Keyword

The default keyword specifies which code to run if there is no case match.


Default does not have to be the last condition in the code block.


Remember:

If default is not the last case in the switch block, remember to end it with a break.

Code Fall-Through

You can use a switch statement if you want multiple cases to use the same code.

    var whatDayIsIt = prompt('What day of the week is it?');
    var text;
    switch(whatDayIsIt) {
      case 'Monday':
      case 'Tuesday':
      case 'Wednesday':
      default: 
        text = 'I wish it was the weekeend.';
        break;
      case 'Thursday':
      case 'Friday':
        text = 'The weekend is so close that I can taste it!'; break;
      case 'Saturday':
      case 'Sunday':
        text = 'Weekends are the best.'; break;
    }
    console.log(text);

Loops

Loops will execute a block of code a number of times.


Loops will run the same code over and over again, and output a different value.

Different Kinds of Loops

  • for - loops through a block of code until the specified condition is false.
  • for/in - loops through the properties of an object. (We will talk about objects next week).
  • while - loops through a block of code while a specific condition is true.
  • do/while - loops through a block of code while a specific condition is true, but it will always go through the loop at least once.

For loops

This is the most common type of loop.

  
    // will count from 1 to 10
    for (var i = 1; i <= 10; i++) {
        console.log(i);
    }
  • Statement 1: (var i = 1;) sets a variable before the loop starts.
  • Statement 2: (i <= 10;) defines the condition for the loop.
  • Statement 3: (i++) is executed each time after the code block ends.

Notice that each statement is separated by a semi-colon

Logical Operators that you will often see in loops


Example Name Result
i++ Increment Adds 1 to i.
i-- Decrement Subtracts 1 from i.

x++ vs. ++x

Postfix

  
    var x = 9;
    var y = x++; 
    // y = 9; x = 10; 
    // because the '++' happens after the assignment (postfix)

Prefix

  
    var a = 9;
    var b = ++a; 
    // b = 10; a = 10; 
    // because the '++' happens before the assignment (prefix)

If you write the addition signs first, you will change both variables.

x-- vs. --x

Postfix

  
    var x = 9;
    var y = x--; 
    // y = 9; x = 8; 
    // because the '--' happens after the assignment (postfix)

Prefix

  
    var a = 9;
    var b = --a; 
    // b = 8; a = 8; 
    // because the '--' happens before the assignment (prefix)

If you write the subtraction signs first, you will change both variables.

Infinite Loops


Beware of these!

Make sure that the condition will be false at some point, or your loop will go on forever...

Loops and logic

You can add other statements or logical operators inside the loops.

  
    //Count from 1 to 50
    for (var i = 1; i <= 50; i++) {
        console.log (i);
        //Says 'Buzz' after multiples of three
        if (i % 3 === 0) {
            console.log (' Buzz');
        }
        //Says 'Bang' after multiples of five
        if (i % 5 === 0) {
            console.log (' Bang');
        }
    }

Using Break in a Loop

  
    //Count from 100 to 200
    for (var i = 100; i <= 200; i++) {
        console.log('Testing ' + i);
        //Stop at the first multiple of 7
        if (i % 7 == 0) {
            console.log('Found it! ' + i);
            break;
        }
    }

While Loop

A while loop will repeat the same code over and over as long as the specified condition is true.

  
    var bottlesOfBeer = 99;
    while (bottlesOfBeer >= 1) {
        console.log(bottlesOfBeer + ' bottles of beer on the wall');
        bottlesOfBeer--;
        // could also write this bottlesOfBeer -= 1; 
        // or bottlesOfBeer = bottlesOfBeer - 1;
    }

Do/While Loop

The do/while loop is a variant of the while loop. The code block will execute once, even if the condition is false, because the code block is executed before the condition is tested.

  
    var text = "";
    var i = 0;
    do {
      text += 'The number is ' + i + '<br />';
      i++;
    } while (i < 10);
    document.write(text);

Don't forget to increase the variable, otherwise the loop will never end.

Comparing for and while

The two loops are very similar.

  
    for(var i = 0; i < 12; i++) {
      console.log(i);
    }
  

  
    var i = 0;
    while(i < 12) {
      console.log(i);
      i++;
    }

The difference is that in a while loop statement 1 and statement 3 are in different places.

Arrays

Arrays are ordered lists of values.

  
    var arrayName = [element0, element1, ...];

You can put different types of data into an array.

  
    var rainbowColors = ['Red', 'Orange', 'Yellow', 'Green',
    'Blue', 'Indigo', 'Violet']; // strings
    var lotteryNumbers = [33, 72, 64, 18, 17, 85]; // numbers
    var myFavoriteThings = ['Broccoli', 1024, 'Sherlock']; // mixed

How to create an array


  • Declare a variable
  • Assign the variable square brackets
  • Inside of the square brackets, add your ordered list, separated by commas.
  • Make sure to end it with a semi-colon
  
    var fruit = ['Kiwi', 'Strawberry', 'Mango'];

Zero Index


Arrays have a zero index, which means that the first item in an array is at index 0. The second item is at index 1. And so on.

  
    var fruit = ['Kiwi', 'Strawberry', 'Mango'];
    // 'Kiwi' is at index 0;
    // 'Strawberry' is at index 1;
    // 'Mango' is at index 2;

Array Length

The length property tells you how many items are in an array

  
    var rainbowColors = ['Red', 'Orange', 'Yellow', 'Green',
    'Blue', 'Indigo', 'Violet'];
    console.log(rainbowColors.length); // output 7;

The length property is always one more than the highest array index, as the length count starts with 1 instead of 0.

Using Arrays

You can access items with bracket notation by using the index of the item. Programmers start counting at zero.

  
    var rainbowColors = ['Red', 'Orange', 'Yellow', 'Green',
    'Blue', 'Indigo', 'Violet'];
    var firstColor = rainbowColors[0]; // Red
    var lastColor = rainbowColors[6];  // Violet

Changing arrays

You can use bracket notation to change an item in an array

  
    var myFavoriteThings = ['Broccoli', 1024, 'Sherlock'];
    myFavoriteThings[0] = 'Asparagus'; 
    // ['Asparagus', 1024, 'Sherlock']

Expanding arrays

Arrays do not have a fixed length. You can use push() to add something to the end of an array.

  
    var myFavoriteThings = ['Broccoli', 1024, 'Sherlock'];
    myFavoriteThings.push('Dancing'); 
    // ['Broccoli', 1024, 'Sherlock', 'Dancing']

Decreasing arrays

You can use pop() to pop off the last element in an array.

  
    var myFavoriteThings = ['Broccoli', 1024, 'Sherlock', 'Dancing'];
    myFavoriteThings.pop(); 
    // ['Broccoli', 1024, 'Sherlock']

Shifting arrays

You can use shift() to remove the first element in an array, and shift all of the other elements up one place.

  
    var myFavoriteThings = ['Broccoli', 1024, 'Sherlock', 'Dancing'];
    myFavoriteThings.shift(); 
    // [1024, 'Sherlock', 'Dancing']

unshifting arrays

You can use unshift() to add an element to the beginning of an array, and shift all of the other elements back one place.

  
    var myFavoriteThings = ['Broccoli', 1024, 'Sherlock', 'Dancing'];
    myFavoriteThings.unshift('Kiwi'); 
    // ['Kiwi', 'Broccoli', 1024, 'Sherlock', 'Dancing']
